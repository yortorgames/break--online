# BREAK!!-Online

**Competitive Block Puzzle Game by YortorGames**

---

## 📱 About

BREAK!!-Online is a competitive block puzzle game for Android where players place colorful blocks on a 10x10 grid, clear full lines, and compete on global leaderboards.

**Features:**
- 🎮 Classic Mode - Practice offline
- 🏆 Ranked Mode - Compete globally
- 📊 Global Leaderboards with 6 rank tiers
- 🔥 Combo System with score multipliers
- 💎 Beautiful neon cyberpunk design
- ⚡ Optimized 60 FPS gameplay
